🎓 MTEC Journal (Электронный журнал студентов)

MTEC Journal — это современное desktop-приложение для ОС Windows, разработанное на C++. Программа предназначена для ведения электронного журнала оценок, управления списками студентов колледжа (или другого учебного заведения) и автоматического подсчета статистики успеваемости.

Интерфейс построен на базе популярной графической библиотеки Dear ImGui с использованием аппаратного ускорения DirectX 11, что обеспечивает быструю и плавную работу приложения.

✨ Основные возможности

🔐 Система авторизации: Два уровня доступа — Администратор (полный доступ к редактированию базы) и Преподаватель (выставление оценок по своему предмету).

👥 Управление студентами: Добавление, редактирование, удаление студентов. Перевод из группы в группу с сохранением истории профильных предметов в "Архив".

📝 Журнал оценок: * Выставление оценок по 10-балльной шкале.

Учет пропусков (Уважительные/Неуважительные/Болезнь).

Указание типа работы (Домашняя, Практическая, Экзамен и т.д.). Экзаменационные оценки имеют больший вес.

Встроенный календарь для выбора даты.

📊 Продвинутая статистика: Автоматический расчет среднего балла за семестр, рейтинг ТОП-10 групп по успеваемости с визуальными прогресс-барами.

🔍 Поиск и фильтрация: Мгновенный поиск студентов по ФИО, фильтрация по группам и сортировка по баллам или алфавиту.

💾 Локальное сохранение: Все данные автоматически и безопасно сохраняются в локальный текстовый файл students_pro.txt.

🛠 Технологии

Язык программирования: C++ (стандарт C++17)

Графический интерфейс: Dear ImGui

Рендеринг: DirectX 11

Платформа: Windows (Win32 API)

🚀 Как запустить проект

Чтобы скомпилировать и запустить проект на вашем компьютере, вам понадобится Visual Studio с установленной поддержкой разработки классических приложений на C++.

Склонируйте репозиторий:

git clone [https://github.com/ВАШ_НИК/НАЗВАНИЕ_РЕПОЗИТОРИЯ.git](https://github.com/ВАШ_НИК/НАЗВАНИЕ_РЕПОЗИТОРИЯ.git)


Откройте проект (файл .sln) в Visual Studio.

Убедитесь, что библиотеки ImGui подключены к проекту (файлы imgui.cpp, imgui_impl_win32.cpp, imgui_impl_dx11.cpp и заголовочные файлы лежат в папке).

Выберите конфигурацию Release / x64 и нажмите F5 (или "Сборка" -> "Собрать решение").

📂 Структура проекта

main.cpp — Точка входа приложения, логика инициализации окна Windows, настройка DirectX 11 и весь код графического интерфейса (ImGui).

Student.h — Заголовочный файл с описанием классов и структур данных (студенты, оценки, логика расчета среднего балла).

students_pro.txt — Файл базы данных (создается автоматически).

Разработано с душой для автоматизации учебного процесса.
